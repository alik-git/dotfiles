# Nautilus Copy-Path Scripts

These two Nautilus scripts are managed by chezmoi and will be installed to `~/.local/share/nautilus/scripts/`.

Files:

- `CopyPath`: copy the selected file path to the clipboard.
- `CopyFolder`: copy the current folder path to the clipboard.

Notes:

1. These scripts depend on `xclip`.
2. After applying them, restart Nautilus with `nautilus -q` if the scripts do not appear immediately.
